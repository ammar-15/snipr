# App Overview

## Purpose

Snipr is a stock call tracker for Twitter traders. Given a Twitter profile URL, the app analyzes recent tweets to identify bullish or bearish calls on stocks. It then evaluates the accuracy of those calls based on current performance of those stocks and assigns a reliability score to each creator. It uses yahoo finance data to validate the stock calls.

## How it works
just enter the twitter profile link of a user and hit 'Go'

## Features

- **Dashboard**: Central hub for analyzing twitter stock sentiments from twitter users.
- **Your Snipes**: A dedicated page for history of older usernames typed in.
- **Search Bar**: A search bar to type in twitter account user links instead of just usernames for stock analysis.
- **User Management**: Admin tools for managing username and password.
- **Authentication**: Secure login and registration system.
- **Analytics**: Uses AI for analyzing tweet sentiments of twitter users against yahoo finance data.

## Navigation

- **Sidebar**: Provides quick access to all major pages.
- **Header**: Displays the app logo and mode toggle.

## Theme (Light/Dark mode)
- There is a light/dark mode toggle in the header (top area of the screen).
- Where to find it:
  - In the header, usually on the top right (theme icon/button).
- What it does:
  - Switches between light mode and dark mode for the app.
