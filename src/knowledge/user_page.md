# User Page

## Path

`/user`

## Purpose

The user page provides tools for managing user accounts.

## Features

- **User List**: Displays all registered users.
- **Actions**:
  - Edit user details.
  - Delete user accounts.
  
## Account settings (User page)
- You can change your **username** and **password** from the **User** page.
- Where to find it:
  - Click your profile/user icon (or “User”) in the top navigation / menu.
- What you’ll see:
  - A section for changing username
  - A section for changing password (or password reset)
